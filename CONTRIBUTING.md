# Contributing

Thank you for considering contributing to Deal Intelligence Starter.

## What we accept

- **New frameworks** -- analytical lenses for different deal dimensions (competitive, stakeholder, timing, etc.)
- **Deployment guides** -- how to run these skills on specific platforms (Cursor, LangChain, CrewAI, etc.)
- **Worked examples** -- pre-filled configurations for different industries or sales methodologies
- **Bug fixes** -- especially in SQL, broken references, or inconsistencies
- **Documentation improvements** -- clearer guides, better examples, troubleshooting

## Guidelines

- Skills must remain harness-agnostic (structured markdown, not tied to one platform)
- Frameworks should use `[YOUR ...]` markers for calibration points
- All CRM/tool references should use `{{PLACEHOLDER}}` syntax
- Keep the collect/classify boundary -- readers collect, frameworks interpret
- Test SQL against the schema before submitting

## How to submit

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Submit a Pull Request with a clear description

## Code of Conduct

This project follows the [Contributor Covenant Code of Conduct](CODE_OF_CONDUCT.md).
